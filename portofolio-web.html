DETAIL PROYEK WEB

SPK Rekomendasi Pembelian Smartphone

Deskripsi Proyek:

Aplikasi web ini adalah implementasi dari skripsi saya, yang dirancang untuk membantu pengguna menemukan smartphone yang paling sesuai dengan kebutuhan mereka. Aplikasi ini menggunakan metode *Simple Additive Weighting* (SAW) untuk menghitung skor setiap alternatif (smartphone) berdasarkan kriteria yang telah ditentukan (seperti harga, kamera, performa, dll.).

Teknologi yang Digunakan:

  • Back-End: PHP Murni (untuk semua logika perhitungan SAW dan manajemen data).
  • Front-End: HTML, CSS, dan JavaScript Murni (untuk antarmuka pengguna dan input).
  • Database: MySQL.

DETAIL PROYEK WEB

SPK Rekomendasi Jurusan (Universitas Perjuangan)

Deskripsi Proyek:

Aplikasi web ini dirancang untuk membantu calon mahasiswa baru di Universitas Perjuangan Tasikmalaya yang masih bingung dalam menentukan pilihan jurusan. Sistem ini menggunakan metode *Simple Additive Weighting* (SAW) untuk menghitung dan memberikan rekomendasi berdasarkan nilai (misal: nilai rapor) dan/atau minat yang diinput oleh pengguna.

Teknologi yang Digunakan:

  • Back-End: PHP Murni (untuk logika perhitungan SAW).
  • Front-End: HTML, CSS, dan JavaScript Murni.
  • Database: MySQL.
